# Python 기초
- print() : 콘솔에 출력하는 '함수'
- type() : 소괄호 내에 있는 데이터(argument)가 어떤 자료형인지 알려주는 명령어

```python
print('python 수업을 시작했습니다.')
print('''
다중으로 작성하고 싶을 때는 \'\'\'\'\'\'으로 작성하는 방법도 있습니다.
''')

print(1)        # 숫자 자료형
print('1')      # 문자열 자료형

print(type(1))      # 결과값 : <class 'int'>

```

- 변수(Variable) : 데이터를 저장하는 바구니

  - float() : 실수 자료형으로 바꿔주는 함수

# Python 조건문
1. if문 : if문은 조건이 참이 때만 해당 블록의 코드 실행
2. if - else 문 : if는 조건이 참일 때 실행 / false 일 때는 else 부분 실행
3. if -elif - else 문

if 조건문의 전체 형식 :
if 조건식1 :
    실행문1
(elif 조건식2 :)
    (실행문2)
(elif 조건식3 :)
    (실행문3)
(else:)
    (실행문4)
()는 optional 입니다.

```python
age = int(input('당신의 나이를 입력하세요 >>> '))

if age > 19 :
    print('성인입니다.')
elif age <= 19 and age > 14 :
    print('청소년입니다.')
elif age <=14 and age > 5 :
    print('어린이입니다.')
else:
    print('유아입니다.')
```

    조건문 내부에 또 다른 조건문을 사용
```python
age = 21
has_ticket = True

if age >= 19 :

```

- 비교 연산자 :
    1) == : 같다
    2) != : 같지 않다
    3) > : 초과
    4) < : 미만
    5) >= : 이상
    6) <= : 이하
- 논리 연산자 :
    1) and : A and B -> A와 B가 모두 참일 때 실행문 실행      &&

    3) nor : if not A - > A가 false일 때 실행문 실행         !

# Python 반복문
1. while 반복문
    while 다음에 나오는 조건식이 참이라면 이하의 실행문이 반복 실행됨(조건문이 False가 될 때까지)
형식 :
while 조건식1 :
    반복실행문

```python
# 1부터 10까지 출력
n1 = 1
while n1 < 11 :
    print(n1)
    n1 += 1     # python에는 ++ 연산자가 없음

# 10부터 1까지 역순으로 출력
n2 = 10
while n2 > 0 :
    print(n2)
    n2 -= 1
```

2. 중첩 while 문 (내부에 while, if 사용 가능)
```python
# 중첩 while문과 f-string을 사용하여 구구단 출력
dan = 2
while dan < 10 :
    number = 1      # 내부에 선언
    while number < 10 :
        print(f'{dan} X {number} = {dan * number}')
        number += 1
    dan += 1
```

3. for 문
형식 :
for i in range (시작값, 종료값, 증감값):
    반복실행문
- i가 0부터 시작
- range() : 몇번 반복할 것인가를 지정하는 함수
          : for문과 함께 연계되어서 사용됨

```python
# 1부터 10까지 출력
for i in range(10) :
    print(i +1)
```

- range() 함수의 응용 : rang((시작값), 종료값, (증감값))
  - 시작값 : 생략가능, 생략하면 0부터 시작
  - 종료값 : 명시하지 않으면 끝까지 진행
  - 증감값 : 생략 가능, 생략할 경우에 1씩 증가
  - 종료값은 '미만'으로 적용됨

```python
for i in range(1, 10, 1) :
    print(i)
```

- range()가 필수는 아님
- default 형태로 작성했을 때의 형식 :
for 변수명 in interable(반복가능객체):
    반복실행문

range() 함수를 쓸 필요 없이 반복 가능 객체(list / tuple / string / set 등)의 처음부터 끝까지 반복 가능

```python
str_example = '안녕하세요'
for i in str_example :
    print(i)

num_list = [1, 2, 3, 4, 5]
for i in num_list :
    print(i, end=' /')
```
- end= ' ' : print() 함수는 자동 개행이기 때문에 마무리를 사용자화하고 싶을 때 사용

# Python index
- 마이너스 인덱스 : 문자열의 뒤에서부터 부여하는 번호. 맨 마지막 데이터의 인덱스 넘버는 -1
```python
str_example = 'hello, python!'
print(str_example[0])   # 결과 : h
print(str_example[1])   # 결과 : e
print(str_example[0])   # 결과 : l
print(str_example[2])   # 결과 : l
print(str_example[3])   # 결과 : o

for alphabet in str_example :
    print(alphabet, end= ' ')

print(str_example[-1])  # 결과 : !
print(str_example[-2])  # 결과 : n
print(str_example[-3])  # 결과 : o

print(str_example[:3:])
# 0번지부터 순서대로 3번 인덱스 미만까지만 출력한다는 의미
```

- 문자열 슬라이싱(slice) : 문자열의 인덱스를 활용하여 한 문자 이상으로 구성된 단어나 문장을 추출할 때 사용하는 방법
    추출하고자 하는 단어나 문장의 시작 인덱스와 종료 인덱스를 통해 그 사이 문자들을 추출할 수 있음.

형식 :
a[시작인덱스 : 종료인덱스 : 증감값]
시작인덱스 : 생략하면 처음부터 추출
종료인덱스 : 생략하면 끝까지 추출
증감값 : 생략하면 1씩 증가함(인덱스 넘버가 0부터 1씩 증가한다는 뜻.)

```python
# 네 자리 숫자를 입력받아 그 숫자의 맨 마지막 자리 숫자 출력
# 조건문으로 맨 마지막 자리 숫자의 홀 / 짝 판별

number = input('네 자리 숫자를 입력하세요 >>> ')
result = ' '

if int(number) % 2 == 0 :
    result = '짝수입니다.'
else :
    result = '홀수입니다.'

print(f'맨 마지막 숫자는 {number[-1]}입니다.')
print(f'맨 마지막 숫자는 {number[-1]}이며, {result}')
```
